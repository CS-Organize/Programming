# Bitwise Operations

| Operation | Symbol | Example | Result |
|-----------|--------|---------|--------|
| AND       | `&`    | `5 & 3` | `0101` & `0011` = `0001` = `1` |
| OR        | `\|`   | `5 \| 3`| `0101` \| `0011` = `0111` = `7` |
| XOR       | `^`    | `5 ^ 3` | `0101` ^ `0011` = `0110` = `6` |
| NOT       | `~`    | `~5`    | ~`0101` = `1010` = `-6` |
| Left Shift| `<<`   | `5 << 1`| `0101` << 1 = `1010` = `10` |
| Right Shift| `>>`  | `5 >> 1`| `0101` >> 1 = `0010` = `2` |
